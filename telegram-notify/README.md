# Telegram Notification Skill

Send Telegram messages with markdown formatting, auto-retry, and rate limiting.

## Features
✅ Send text notifications with Telegram markdown
✅ Auto-retry (max 3, configurable)
✅ Rate limiting (configurable)
✅ Message history
✅ Connection testing
✅ Logging
✅ Cross-runtime (Claude, Copilot, Gemini)
